# Online shopping mall api

![API-tree](api-tree.png)

- routes : URL 라우터
- services : 비즈니스 로직
- models : DB 스키마
- repositories : DB의 interface

## Quck Start

```bash
git clone https://github.com/passwd10/shopping-mall-api.git
npm install
npm start
```

## 사용 기술

- Node.JS
- express
- npm
- MongoDB
- Docker
